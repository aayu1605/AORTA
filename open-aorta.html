AORTA

This launcher can show the static build when available.

Option A: View static build

If you already built the app, it will load below. If you see a blank area, run the build steps under Option B.

Option B: Start the dev server

  1. Open a terminal.
  2. Run: cd frontend/aorta-frontend
  3. Install deps (first time): npm install
  4. Start dev server: npm run dev

After the server starts, open the site in your browser.
